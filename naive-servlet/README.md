1. 切换到本模块顶层目录下
2. 执行 `mvn clean jetty:run`,
3. 之后通过浏览器访问 [http://localhost:8080/naive-servlet/any](http://localhost:8080/naive-servlet/any) (url中的 `any` 可以替换为其他内容)
